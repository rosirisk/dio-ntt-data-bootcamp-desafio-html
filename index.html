O QUE É, E, PARA QUE SERVE?

(Voltar)

O HTML (HyperText Markup Language) é uma linguagem de marcação de hipertexto. Ele se popularizou através de Tim Berners-Lee, este é considerado o pai da internet, foi ele quem popularizou tanto o HTTP quanto o HTML, e a primeira página Web foi ao ar em 06/08/1991. A sua função é criar páginas Web, e, para isto irá utilizar de Tags.

TAGS

(Voltar)

As tags são interpretadas pelo navegador através das instruções, para que assim apresente o seu conteúdo ao usuário.

ALGUMAS TAGS

(Voltar)
  • <h1> até <h6>: São utilizadas para cabeçalho ou título, ela é dividida por níveis, como por exemplo título, subtítulo, e assim sucessivamente. Sendo <h1> o maior nível e <h6> o menor nível.
  • <p> É o corpo do texto, paragraph(parágrafo) o uso é para quebra de linha ou parágrafo novo.
  • <mark> Esta funciona como um marca texto.
  • <small> vai deixar o texto menor.
  • <i> Irá deixar as letras mais inclinadas (ficará em itálico).
  • <u> (underline) deixa o texto sublinhado.
  • <strong> Deixará o texto em negrito.
  • <ol> É a tag de lista ordenada, ela precisa das filhas para funcionar, <li> lista (list item), o modo que irei colocar por exemplo item 1, item 2 e assim sucessivamente.
  • <ul> Esta é a tag de lista não ordenada, neste caso, ao inveś de ser em números, e, ter uma ordem, usará marcadores no lugar dos números.
  • <li> Significa lista.
  • <a> anchor, é uma âncora, ao clicar vai direcionar dentro da sua página, ir de um lugar para outro dentro da página, tem um atributo específico é o href, ele direciona para onde está indo o site (abre na mesma aba), faz a ligação de uma página externa
  • <hr> Vai traçar uma linha abaixo, como se fosse uma linha separando por exemplo.
  • <sub> De acordo com o que voce define, letra ou o número, irá aparecer abaixo da linha.
  • <sup> Com essa Tag a letra ou o número ficará sobre a linha, marca de registrado, ou então, número elevado, por exemplo, uma nota de rodapé.
  • <bloquote> Quando vou colocar uma citação, ou, quando vou colocar uma observação utilizo essa tag.
  • <font> Essa tag define cor, tamanho e estilo da fonte, entretando, ela não é mais utilizada no HTML5 , e para alterar as caracteristicas do texto utiliza-se o CSS.
  • <del> Irá definir o texto que foi excluído, onde, colocará uma linha sobre o texto que foi excluído.
  • <abbr> Essa tag é utilizada quando tenho uma abreviação ou sigla, e se utilizada com o atributo title irá mostrar a explicação ao passar o mouse sobre a sigla ou abreviação.
